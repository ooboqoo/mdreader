# highlight.js

The official site / 官网： https://highlightjs.org   
GitHub 地址：https://github.com/isagalaev/highlight.js

Highlight.js is a syntax highlighter written in JavaScript. It works in the browser as well as on the server. It works with pretty much any markup, doesn’t depend on any framework and has automatic language detection.

Highlight.js 是用 JavaScript 编写的语法高亮插件，可以工作在浏览器和服务器上。它支持高亮众多不同语言，且不依赖任何框架，并能自动侦测语言类型。

### Download a custom bundle / 自定义插件包

地址：https://highlightjs.org/download/

You can download a custom bundle including only the languages you need.   
在下载页面，您可以自由选择要高亮的语言类型，选好后点下方的 downlaod 就可下载自定义的 highlight.pack.js 文件。
